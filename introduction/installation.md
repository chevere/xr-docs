# Installation

## System Requirements

XR Debug is built and tested for unix-like systems. It depends on `chevere/chevere` and it requires PHP.

* [PHP 8](https://www.php.net/releases/8.0)
* [chevere/chevere](https://github.com/chevere/chevere)
